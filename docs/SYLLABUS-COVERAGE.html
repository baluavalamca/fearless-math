FearlessMath — Syllabus Coverage Map

How the current authored lessons line up against the NCERT / CBSE primary-maths syllabus (Class 1 → Class 5). Prepared July 2026.

✅ Covered 🟡 Partial — taught at a different class, or only one slice of the topic ❌ Missing at this class
How to read this. The app organises lessons by topic & strand, while NCERT's newer NEP books (Joyful Mathematics for Class 1–2, Maths Mela for Class 3–4) bundle topics under thematic chapter names. So this map compares at the topic level, not chapter titles. "Partial" usually means the idea is taught in FearlessMath, just at an adjacent class (the app teaches a topic once and lets it carry forward, whereas NCERT re-spirals many topics every year).

Bottom line: the 54 existing lessons are genuine, grade-appropriate CBSE topics — not placeholders — but they're a curated spine (~13–14 per class), not the full year. The clearest structural gaps are Data Handling and Patterns (each authored at only one class though NCERT teaches them every year), geometry & measurement in the Foundation years, and 4–5 digit numbers at Class 4.
